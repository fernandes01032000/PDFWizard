═══════════════════════════════════════════════════════════

📝 MODO DESIGN (Criar Templates):

   1. Faça upload de um PDF
   2. Clique nos tipos de campo (esquerda)
   3. Arraste os campos para posicionar no PDF
   4. Digite um nome e clique em "Salvar Template"

═══════════════════════════════════════════════════════════

✍️ MODO PREENCHIMENTO (Usar Templates):

   1. Clique na aba "Modo Preenchimento"
   2. Selecione um template salvo
   3. Preencha os campos (veja preview em tempo real)
   4. Clique em "Gerar e Baixar PDF"

🎨 TIPOS DE CAMPO DISPONÍVEIS:

   📝 Texto (linha única)      - Nome, CPF, matrícula
   📄 Texto (múltiplas linhas) - Observações, descrições
   ☑️ Checkbox                 - Sim/Não, marcações
   📅 Data                     - Datas
   ✍️ Assinatura               - Nome para assinar

🔧 PROBLEMAS COMUNS:

      → Verifique se é um PDF válido

   ❌ Templates não aparecem
      → Certifique-se de ter salvado no Modo Design
      → Verifique o console (F12)

   ❌ Campos não ficam no PDF
      → Posicione os campos corretamente
      → Preencha todos os campos obrigatórios
